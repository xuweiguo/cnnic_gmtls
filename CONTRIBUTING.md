# Contributing

Thanks for your interest in contributing to this project.

## Development

- Go version: 1.21+
- Run tests: `go test ./...`

## Style

- Use `gofmt` for all Go files.
- Keep exported identifiers in `CamelCase` and unexported in `mixedCase`.

## Pull Requests

Please include:
- Summary of changes
- How to run tests (if applicable)
- Any protocol-impact notes
